# Messaging Spec — Dual Waste→Gain Narrative (all 4 apps)

## The core inequality (yours)
Each app is a small recurring fee that prevents TWO large, frequent wastes:
  MONEY — $ spent on things you abandon (unfinished books, unworn clothes,
          poured-out drinks, unplanned delivery).
  TIME  — hours lost searching/deciding, with no guarantee of a good result.

Perceived Value > Cost  →  the value is BOTH: money saved AND time reclaimed.

## The transformation arc (must appear across the whole flow, not just paywall)
For each app, every stage moves the user along BOTH axes:

  MONEY:  "money wasted on X you didn't want"  →  "money spent only on X you'll love"
  TIME:   "hours lost searching/guessing"      →  "a match in ~10 seconds → hours back
                                                    for the thing you actually enjoy"

The effort trade the user makes: invest ~5 minutes once (rate / save / review) →
the app learns your taste → you save money AND hours every week after.

## Where each axis shows up (4 touchpoints)
  1. WELCOME — name BOTH wastes (money line + time line).
  2. A Y/N QUESTION — make the user self-identify the waste (one money-flavored,
     optionally one time-flavored).
  3. BRIDGE + REVEAL — demonstrate the TIME win ("in 10 seconds, here's your match").
  4. PAYWALL — headline (outcome) + bullets (each bullet ideally hits money OR time)
     + a MONEY-math line + a TIME-math line, side by side.
